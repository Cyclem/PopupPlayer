# 伪装广告弹窗播放器 (Ad Popup Player)

这是一个高度伪装成“网页牛皮癣广告”的本地视频播放器。程序采用无边框设计，UI 交互完全隐藏在广告背景图（`ad_bg.png`）的特定像素区域内。

## 核心功能

- **界面伪装**：采用抗锯齿圆角与无边框设计，视觉上与普通的页游弹窗广告无异。
- **全局置顶**：窗口始终处于系统最前端，默认停靠于屏幕右下角。
- **交互隐藏**：无可见的播放/暂停按钮或控制栏，点击背景图特定位置触发相应功能。
- **高级缩放**：支持通过拖拽左上角进行等比缩放，缩放时右下角坐标作为锚点固定不动。
- **状态记忆**：自动创建本地 `config.ini` 文件，记录窗口位置，缩放比例和每个视频的断点时间记录，下次启动或者播放视频自动还原。

## 播放器特性

- **实时预览**：拖动进度条滑块时，视频画面随之实时跳动，并且在弹窗上方显示当前/总时长，实现进度预览。
- **自动静音**：拖动进度条过程中自动静音，防止产生杂音，松开后恢复。
- **隐形暂停**：点击视频画面区域即可切换播放或暂停状态。
- **字幕支持**：自动识别同名 `.srt` 或 `.ass` 字幕，支持双语垂直堆叠显示，并具备动态字号缩放功能，防止长句割裂。

## 字幕使用规范

1. **自动加载**：字幕文件需与视频文件位于**同一文件夹**下。
2. **命名规则**：字幕主文件名需与视频文件名**完全一致**（如 `1.mp4` 对应 `1.srt` 或 `1.ass`）。
3. **解析策略**：程序会自动剥离 `.ass` 文件中的复杂特效代码，提取纯净文本进行渲染，以保证在小窗口下的可读性。

## 编译与打包

### 环境要求

- Python 3.10+
- 依赖库：`pip install PyQt6`

### 打包为 EXE (Windows)

请确保 `main.py` 和 `ad_bg.png` 在同一目录，运行以下命令：

```bash
pyinstaller --noconsole --onefile --add-data "ad_bg.png;." main.py
```

生成的可执行文件位于 `dist` 文件夹内。程序运行后会在同级目录产生 `config.ini` 配置文件，删除该文件可重置软件状态。
