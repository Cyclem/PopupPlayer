# 伪装弹窗广告摸鱼播放器

这是一个高度伪装成“网页牛皮癣广告”的强力本地/网络视频播放器。程序采用无边框设计，UI 交互完全隐藏在广告背景图（`ad_bg.png`）的特定像素区域内。最新v2.0版本底层由强大的 **MPV** 引擎驱动，支持海量格式、B站直连解析以及纯正的字幕特效渲染。

## ✨ 核心功能

- **界面伪装**：采用抗锯齿圆角与无边框设计，视觉上与普通的页游弹窗广告无异。
- **全局置顶**：窗口始终处于系统最前端，默认停靠于屏幕右下角。
- **一键 B站 播放**：自动识别系统剪贴板中的 Bilibili `BV` 号，无需繁琐输入，点击即看。
- **交互隐藏**：无可见的常规播放/控制面板，所有功能均通过点击背景图的“伪装区域”触发。
- **高级缩放**：支持通过拖拽左上角进行等比缩放，缩放时右下角坐标作为锚点固定不动。
- **状态记忆**：自动创建本地 `config.ini` 文件，记录窗口位置、缩放比例，以及每个本地/网络视频的**断点续播**记录，下次播放自动无缝还原。

## 🎬 播放器特性

- **MPV 强力驱动**：告别系统自带解码器限制，利用 MPV 提供全格式视频支持及流畅的硬件解码。
- **关键帧实时预览**：拖动进度条滑块时，实现丝滑的画面实时预览，不卡顿。
- **原生特效字幕**：完美支持 `.ass` 复杂特效字幕。
- **隐形暂停**：点击视频画面区域即可随时切换播放或暂停状态。

## 📜 使用须知

1. **本地字幕自动加载**：字幕文件需与视频文件位于**同一文件夹**，且主文件名**完全一致**（如 `1.mp4` 对应 `1.srt` 或 `1.ass`）。MPV 会自动加载并渲染。
2. **B站解析 (Cookies)**：
   - 没有 `cookies.txt` 的情况下，B站解析大概率失败。
   - 请在 `.exe` 程序所在同级目录下创建一个名为 `cookies.txt` 的文件，并将你在 Bilibili 登录后的 Cookie 按照 Netscape 格式填入其中。程序会自动读取并应用于 `yt-dlp` 解析。

## 🛠️ 编译与打包

### 1. 环境要求

- Python 3.8+ (推荐 64位)

- 依赖库安装：

  ```bash
  pip install PyQt6 python-mpv pyperclip yt-dlp
  ```

- **核心组件 (Windows 必备)**：
  因为使用了 MPV 引擎，请前往下载 `mpv-1.dll` (64位版本)，并确保该 DLL 文件放置在 `main.py` 的**同级目录**下，否则程序将无法启动。

### 2. 打包为 EXE (Windows)

请确保 `main.py` 、 `ad_bg.png`、`yt-dlp.exe`和`mpv-1.dll` 在同一目录下，运行以下 PyInstaller 命令：

```bash
pyinstaller --noconsole --onefile \
--add-data "ad_bg.png;." \
--add-data "mpv-1.dll;." \
--add-data "yt-dlp.exe;." \
main.py
```

**mpv-1.dll获取方式:**

*   **下载地址**：[mpv-winbuild-cmake (shinchiro)](https://github.com/shinchiro/mpv-winbuild-cmake/releases)
*   **找哪个文件？**：
    找文件名包含 `mpv-dev-x86_64-XXXX.7z` 的文件（因为你的 Python 3.14 是 64 位的，必须下载 **x86_64** 版本）。
*   **如何提取**：
    下载解压后，在 `lib` 文件夹或者根目录下就能找到 `mpv-1.dll`。


### 3. 发布须知

生成的可执行文件位于 `dist` 文件夹内。
*(程序运行后会在该目录自动生成 `config.ini` ，删除 `config.ini` 可重置软件全部状态)*
