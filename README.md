# 伪装弹窗广告摸鱼播放器

这是一个伪装成“网页牛皮癣广告”的强力本地/网络视频播放器。采用无边框设计，UI 交互完全隐藏在广告背景图中。底层由 **MPV** 引擎驱动，支持海量本地格式、B站直连解析，**新增平台直播模式**。

##  核心特性

| 模块分类     | 功能说明                                                     |
| :----------- | :----------------------------------------------------------- |
| **界面伪装** | 无边框、抗锯齿圆角设计，纯正“页游弹窗广告”视觉。全局置顶，默认停靠屏幕右下角。 |
| **智能嗅探** | 自动识别剪贴板的 **B站 BV号** 及 **主流直播间链接**（B站之外的直播平台不稳定）。 |
| **强力引擎** | 搭载 MPV 底层，告别系统解码器限制。支持完美硬件解码与 `.ass` 复杂特效字幕原生渲染。 |
| **无痕交互** | 无常规播放控制条，所有操作基于“隐形热区”。支持等比缩放、位置记忆与点播断点续播记忆。 |

所有常规交互均通过隐藏热区（左键/拖拽）触发：

| 触发区域                | 动作 | 对应功能                                                     |
| :---------------------- | :--- | :----------------------------------------------------------- |
| **【开始游戏】字样**    | 点击 | 若剪贴板有 `BV号` 或 `直播链接` 则自动加载；否则弹出本地文件选择器。 |
| **视频画面区域**        | 点击 | **点播模式**：切换 播放 / 暂停 状态。<br>**直播模式**：彻底切断直播流 / 再次点击重新加载最新流。 |
| **视频底部金色光点**    | 拖拽 | 调节播放进度，支持关键帧平滑预览（*注：直播模式下自动隐藏*）。 |
| **【广告】字样 (右下)** | 点击 | 停止播放并隐藏画面，瞬间恢复为纯广告背景状态（老板键）。     |
| **【 X 】图标 (右上)**  | 点击 | 保存当前窗口状态与视频断点进度，彻底退出程序。               |
| **窗口左上角区域**      | 拖拽 | 以窗口右下角为固定锚点，进行等比例缩放大小。                 |
| **非功能区空白处**      | 拖拽 | 自由移动窗口在屏幕上的位置。                                 |

##  进阶使用须知

| 注意事项         | 详细说明                                                     |
| :--------------- | :----------------------------------------------------------- |
| **本地字幕**     | 字幕文件需与视频 **同名且在同一文件夹**（如 `1.mp4` 对应 `1.srt` 或 `1.ass`），MPV 将自动挂载渲染。 |
| **Cookies 配置** | 根目录放置 `cookies.txt`可解锁 **B站视频播放** 及绕过部分直播平台的强制验证。 |
| **组件更新**     | 若未来某天网络直播解析失效，通常是平台接口变动。只需去下载最新版 `yt-dlp.exe` 替换同级目录下的旧文件即可修复。 |

##  编译与打包

### 1. 基础环境

- **要求**：Python 3.8+ (推荐 64位)
- **依赖库**：`pip install PyQt6 python-mpv pyperclip yt-dlp`

### 2. 必备前置文件

在执行打包前，请确保 `main.py` 的同级目录包含以下 3 个核心文件：

1. **`ad_bg.png`**：伪装用广告背景图。
2. **`yt-dlp.exe`**：用于网络链接解析。
3. **`mpv-1.dll`** (64位)：MPV核心库。[点击下载 (找 `mpv-dev-x86_64` 压缩包提取)](https://github.com/shinchiro/mpv-winbuild-cmake/releases)

### 3. 一键打包命令 (Windows)

```bash
pyinstaller --noconsole --onefile \
--add-data "ad_bg.png;." \
--add-data "mpv-1.dll;." \
--add-data "yt-dlp.exe;." \
main.py
```

*生成的可执行文件位于 `dist` 文件夹内。首次运行会在同目录自动生成 `config.ini` 文件，若需重置所有设置（位置、缩放、历史记录），删除该 `ini` 文件即可。*
