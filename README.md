# 伪装广告弹窗播放器 (PopupPlayer)

这是一个伪装成页游广告的本地视频播放器。程序采用无边框、无标准UI控件的设计，默认置顶于屏幕右下角。

## 核心功能

- **界面伪装**：无系统标题栏与边框，UI 交互完全依赖底层背景图（`ad_bg.png`）。
- **置顶播放**：窗口全局置顶，默认停靠于屏幕右下角。
- **字幕支持**：自动读取并渲染外挂 `.srt` 格式字幕。
- **状态记忆**：自动生成 `config.ini`，记录上次关闭时的窗口坐标与尺寸，下次启动自动恢复。
- **锚定缩放**：支持通过拖拽左上角进行等比缩放，缩放时右下角坐标保持不变。

---

## 交互说明

本播放器隐藏了常规播放器的操作按钮，所有交互均通过点击背景图的特定区域触发：

| 操作区域 | 触发功能 | 说明 |
| :--- | :--- | :--- |
| **【开始游戏】字样** | 载入并播放 | 点击图片下方的“开始游戏”，弹出文件选择器。选中视频后开始播放。 |
| **【广告】字样** | 停止并重置 | 点击右下角的“广告”二字，视频画面关闭并停止播放，恢复至初始状态。 |
| **【右上角 X】** | 退出程序 | 点击图片右上角的关闭图标，程序将保存当前位置和尺寸参数，随后退出。 |
| **左上角区域** | 等比缩放 | 将鼠标移至窗口左上角（光标变为缩放箭头），按住左键拖拽可等比调整窗口大小。 |
| **其他空白处** | 移动窗口 | 按住非上述功能区的任意位置，拖拽鼠标可移动窗口位置。 |

## 字幕使用规范

程序支持自动加载字幕，但需满足以下条件：

1. **同名且同目录**：字幕文件必须与视频文件放在同一个文件夹下，且主文件名完全一致（例如 `video.mp4` 与 `video.srt`）。
2. **格式要求**：仅支持标准 `.srt` 格式。
3. **编码要求**：如遇中文乱码问题，请将 `.srt` 文件另存为 `UTF-8` 编码格式。

## 编译与打包指南

如需基于源码自行打包为 `.exe` 可执行文件，请参考以下步骤：

### 1. 环境准备
需要 Python 3.10 及以上环境，安装依赖包：
```bash
pip install PyQt6 PyQt6-Qt6 PyQt6-Multimedia PyQt6-MultimediaWidgets pyinstaller
```

### 2. 打包命令 (Windows环境)
请确保 `main.py` 与 `ad_bg.png` 位于同一目录下。在终端中运行以下命令：
```bash
pyinstaller --noconsole --onefile --add-data "ad_bg.png;." main.py
```
